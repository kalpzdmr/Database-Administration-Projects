# Database Administration Projects

## Overview

This repository contains a collection of SQL Server Database Administration projects developed using the **AdventureWorks2022** and **WideWorldImporters** sample databases. The projects focus on essential database administration topics including backup and recovery, security and access control, and performance optimization.

These implementations demonstrate practical SQL Server administration skills and provide hands-on experience with real-world database management tasks.

---

## Project 1: Database Backup and Disaster Recovery Plan

### Description

This project demonstrates how to protect a database against data loss by implementing a comprehensive backup and recovery strategy.

### Implemented Features

* Full Database Backup
* Differential Backup
* Transaction Log Backup
* Backup Verification
* Disaster Recovery Scenario
* Database Restore Operations
* Recovery Model Configuration
* Data Recovery Validation

### Dataset

* AdventureWorks2022

### Technologies

* SQL Server
* T-SQL
